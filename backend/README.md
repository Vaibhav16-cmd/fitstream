# FitStream Backend

This backend powers the fitness and media features of FitStream.

## Main folders

- `src/config`: environment and database setup
- `src/controllers`: request and response handlers
- `src/middleware`: auth, validation, and errors
- `src/models`: MongoDB data blueprints
- `src/routes`: API endpoints
- `src/services`: reusable business logic
- `src/utils`: helper functions
- `src/validations`: request input checks
- `src/seed`: starter workouts and songs

## Run

```bash
npm run dev
```
