# FitStream Frontend

This frontend will contain the dashboard, workout screens, progress views, and media player UI.

## Main folders

- `src/app`: root app setup
- `src/components`: reusable UI pieces
- `src/features`: fitness and media modules
- `src/pages`: route-level screens
- `src/services`: frontend API calls
- `src/store`: shared state like auth and player
- `src/styles`: global styling
